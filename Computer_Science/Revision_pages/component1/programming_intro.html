Introduction to Programming

Tell the computer what to do!

Procedural languages

A procedural language is one where instructions (in the form of source code) are carried out one at a time ie. procedural languages are sequential. Procedural languages are imperative.

Program flow

The path followed by a program is known as program flow. Programs will generally start at the first instruction and end at the last ie. sequential. Even with interrupts and loops, a program will always have a beginning and an end. Program flow may be illustrated in a flow diagram.

Variables and Constants

A variable is used to store values in memory which can change as the program runs. A variable is given a name or ('identifier') which is used to refer to it. A constant is a value which cannot be modified as the program runs eg. PI.

Procedures

A procedure is a section of computer code that performs a specific task. There are two main types: Subroutines and functions. A subroutine is a procedure that can take zero or more values as input (parameters), act upon that data, then return zero, one or more values. A subroutine is always 'called'. Calling a subroutine means the calling procedure sets up the CPU to carry out the instructions within the subroutine. A function is similar to a subroutine but it only ever returns one value.

Strings

Strings start at 0 eg. "HI", at position 0 is "H". Represented by "" in most languages. Strings represent text in a program. Strings can be concatenated by 'adding' two strings together. Strings also have a length which can be useful to dynamically operate on the string. Len(string_name) will return the length of the string. Strings can also be converted to upper and lower case by using .upper() or .lower(). Parts of a string can be replaced using .replace("stufftoreplace", "replaceitwith"). Strings can also be sliced in order to use only the parts required using string_name[start:end]. If no start is specified, then the string will be sliced until the end specified. If no end is specified, the string from the start point all the way to the end will be returned.

File handling

Files are used to store values which one may want to retain eg. scores. f = open(“highscores.txt”,”r”) would allow one to open a file in read only mode. w = write, r = read, a = append etc. lines = f.readlines() reads all of the lines of a file into an array which one can then access. When one has finished with a file, it is important close the input stream eg. f.close(). To write to a file: f.write(“\nJames, 9720”) would be one way; note - it must be opened in a mode which permits writing!
